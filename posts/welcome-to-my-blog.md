# 欢迎来到我的博客

欢迎访问我的个人博客！这是我使用 Next.js 构建的现代化博客平台，具有以下特色功能：

## 主要功能

### 🌓 深色/浅色主题切换
支持深色和浅色主题切换，为您提供舒适的阅读体验。

### 🔍 全文搜索
强大的搜索功能，可以搜索文章标题、内容和标签。

### 🏷️ 标签系统
完善的标签分类系统，帮助您快速找到感兴趣的内容。

### 📱 响应式设计
完美适配桌面端、平板和移动设备。

## 技术栈

这个博客使用了以下技术：

- **Next.js 15** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 现代化样式
- **Markdown** - 文章内容格式
- **Lucide React** - 图标库

## 博客内容

在这里，我会分享：

1. **技术文章** - 编程技巧、最佳实践
2. **学习笔记** - 学习过程中的心得体会
3. **项目经验** - 实际项目中的问题和解决方案
4. **生活感悟** - 技术之外的思考和感悟
